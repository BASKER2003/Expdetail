Definition and Usage

The <li> HTML element is used to represent an item in a list.

It must be contained in a parent element: an ordered list (<ol>), an unordered list (<ul>), or a menu (<menu>).

In menus and unordered lists, list items are usually displayed using bullet points.

In ordered lists, they are usually displayed with an ascending counter on the left, such as a number or letter.

Definition and Usage

The <ol> HTML element represents an ordered list of items typically rendered as a numbered list.

Attribute Type:

Sets the ordering type:

a - for lowercase letters.
A - for uppercase letters.
i - for lowercase Roman numerals.
I - for uppercase Roman numerals.
1 - for numbers (default)

Definition and Usage

The <ul> HTML element represents an unordered list of items, typically rendered as a bulleted list.

Attribute Type:

Sets the ordering type:

circle
disc
square

NOTE:

A fourth bullet type has been defined in the WebTV interface, but not all browsers support it:triangle

Definition and Usage

The <dl> HTML element represents a description list.

This element encloses a list of groups of terms (specified using the <dt> element) and descriptions (provided by <dd> elements).

Common uses for this element are to implement a glossary or to display metadata (a list of key-value pairs).

Definition and Usage

The <dt> HTML element specifies a term in a description or definition list, and as such must be used inside a <dl> element.

It is usually followed by a <dd> element;

However, multiple <dt> elements in a row indicate several terms that are all defined by the immediate next <dd> element.

Definition and Usage

The <dd> tag is used to describe a term/name in a description list.

The <dd> tag is used in conjunction with <dl> (defines a description list) and <dt> (defines terms/names).

Inside a <dd> tag you can put paragraphs, line breaks, images, links, lists, etc.
